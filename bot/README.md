# بوت الفلترة الطبية الذكي — دليل الإعداد

## المتطلبات الأساسية

قبل تشغيل البوت، تحتاج:

### 1. الحصول على API_ID و API_HASH
1. اذهب إلى [my.telegram.org](https://my.telegram.org)
2. سجل دخول برقم هاتفك
3. اختر **API development tools**
4. أنشئ تطبيقاً (أي اسم) — ستحصل على `api_id` و `api_hash`

### 2. الحصول على BOT_TOKEN
1. في تيليجرام، ابحث عن [@BotFather](https://t.me/BotFather)
2. أرسل `/newbot`
3. اتبع التعليمات لاختيار اسم ومعرف للبوت
4. ستحصل على التوكن (token)

### 3. الحصول على OWNER_ID (معرفك الشخصي)
1. ابحث عن [@userinfobot](https://t.me/userinfobot) في تيليجرام
2. أرسل `/start`
3. سيعطيك رقمك الشخصي (User ID)

---

## خطوات التثبيت

```bash
# انتقل لمجلد البوت
cd bot

# ثبت المكتبات
pip install -r requirements.txt

# أنشئ ملف الإعدادات
cp .env.example .env

# عدّل .env وضع بياناتك
```

### محتوى ملف `.env`:
```
API_ID=1234567
API_HASH=abcdef1234567890abcdef
BOT_TOKEN=1234567890:AAExxxxxxxxxxxxxxx
OWNER_ID=987654321
```

---

## تشغيل البوت

```bash
python main.py
```

ثم في تيليجرام، ابحث عن بوتك وأرسل `/start`
